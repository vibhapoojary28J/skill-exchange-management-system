# skill-exchange-management-system

A web-based Skill Exchange Management System developed using Flask, SQLite, HTML, CSS, and JavaScript that enables users to register, manage profiles, showcase skills, connect with others, exchange knowledge, and interact through messaging while implementing DBMS concepts such as CRUD operations.

## Features

* User Registration and Authentication
* Skill Management and Showcase
* Messaging System
* Connection Management
* Analytics Dashboard
* CRUD Database Operations

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS

## Project Structure

* Backend Development using Flask
* Frontend using HTML, CSS, and JavaScript
* Database Management using SQLite
* Integrated Messaging and Dashboard Features
